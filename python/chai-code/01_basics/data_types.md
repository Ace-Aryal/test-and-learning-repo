# Object Types / Data Types

- Numbers : 1234, 3.1415, 3+4j, ob111, Decimal(), Fraction() 
- String : 'str', "That's", b'a\x01c'
- List : [1, [2, "Hello], 3.5] , list(range(10))
- Tuple : (1, "spam, 4, u) , tuple("spam"), namedtuple
- Dictionary : {
    'name' : "Dipesh",
    'age' : 19
}, dict(hours=10)
- Set : set(abc), {'a', 'b', 'c'}
- File: open('eggs.txt'), open(r'C:\ham.bin','wb')
- Boolean : True, False
- None: None
- Functions, Modules, Classes,
- Advanced : Decorators, Generators, Iterators, Metaprogramming
